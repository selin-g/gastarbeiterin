Web-Archive und das digitale Vermächtnis

Das wohl bekannteste Webarchive ist das gemeinnützige Projekt Internet Archive (IA). Mit dem Auf- und Ausbau einer digitalen Bibliothek werden digitale Artefakte wie Bücher, Bilder oder Audiodateien im Internet gespeichert und frei verfügbar gemacht. Einen signifikanten Beitrag zur Konservierung des Webs leistet ihr Online-Dienst Wayback Machine1. Mithilfe von automatisierten Web-Crawlern generieren und speichern sie Snapshots, also Kopien beziehungsweise Momentaufnahmen von Webseiten, die zu einem späteren Zeitpunkt abruf- und rekonstruierbar sind.2

Archivierungsanalyse

Die Archivierungsanalyse untersucht mithilfe der Wayback Machine, wie häufig und über welches Zeitfenster die URLs archvieirt wurden – auch Captures genannt. Die lila Balken zeigen die Anzahl der Captures pro Jahr; graue Balken stehen für Jahre ohne Archivierung. Die Zeitachse beginnt jeweils kurz vor dem Veröffentlichungsjahr des Artikels und erstreckt sich bis in das gegenwärtige Jahr 2026. Damit soll sichtbar gemacht werden, wie lange ein Artikel bereits existierte, bevor er erstmals archiviert wurde.

cdxTimeline = (data, startYear, endYear = 2026) => {
  const years = d3.range(startYear, endYear + 1);
  const counts = new Map(years.map(y => [y, 0]));
  const rows = (data.length > 0 && Array.isArray(data[0]) && data[0][0] === "timestamp")
    ? data.slice(1) : data;
  rows.forEach(d => {
    const y = +d[0].slice(0, 4);
    if (counts.has(y)) counts.set(y, counts.get(y) + 1);
  });
  return years.map(year => ({ year, count: counts.get(year) }));
}

Befund

Aus den obigen Ergebnissen lässt sich weder ein repräsentativer noch kohärenter Befund darüber liefern, nach welchen Schemata die Artikeln archiviert wurden – zumal die Ergebnisse deutliche Unterscheidungen aufweisen. Es ist davon auszugehen, dass die Archivierpraktiken keiner thematischen oder sprachlichen Logik folgen, sondern vielmehr auf institutioneller Ebene stattfinden. Dies lässt sich daraus ableiten, wie unterschiedlich die Häufigkeiten sowie die Zeitintervalle der Archiviervorgänge je Institution sind.

Die Bundeszentrale für politische Bildung sowie der IG Metall präsentieren sich in der Analyse als beispielgebende Institutionen, deren Artikel regelmäßig in der Wayback Machine archiviert wurden. Dies wirft die Frage auf, wieso die restlichen Institutionen – die Friedrich-Ebert-Stiftung und der DGB aufgrund der jungen Veröffentlichungsdaten ausgenommen – Unregelmäßigkeiten, Unvollständigkeiten, und im Fall von ver.di gar eine Nichtexistenz in ihren Archivierpraktiken aufweisen. Es lässt sich nur mutmaßen, ob es sich hier nicht etwa um ein strukturelles Problem handelt, wonach Institutionen keine Verantwortung für das gesellschaftliche Erinnern durch Web-Archivierungen übernehmen.

Schlussendlich ist es aber auch notwenig zu unterstreichen, dass die hier durchgeführte Archivierungsanalyse lediglich einen kleinen, selektiv-kuratierten Einblick in die Archivierpraktiken von acht (beziehungsweise neun) Artikeln aus zehn institutionellen Webpräsenzen bietet. Ob und wie die Gastarbeiterin im breiteren Web-Gedächtnis vertreten ist, lässt sich daraus nicht ableiten.

Analytische Limitation

Neben Web-Crawlern gibt es noch das integrierte Tool Save Page Now von IA „that allows users to capture and preserve web content”3. Damit ermöglicht das IA die Liberalisierung der Webarchivierung, indem sowohl Institutionen als auch Individuen aktiv daran mitwirken können. Dieser interessante Aspekt könnte im Rahmen weiterer Analysen nicht nur Aufschluss über die Akteur:innen geben, sondern auch epistemische Einblicke darin bieten, welche Webseiten und Informationen als besonders ‚archivierungswürdig‘ empfunden wurden.
